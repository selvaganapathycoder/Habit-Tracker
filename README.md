## 🚀 Features

- **Daily Tracking**: Keep tabs on your daily routines easily.
- **Streaks System**: Visually rewarding streaks to keep you motivated (`🔥`).
- **Activity Heatmap**: A GitHub-style contribution graph to see your consistency over the past weeks.
- **Pro UI / UX**:
  - Glassmorphic translucent cards 
  - Vivid gradient text and progress bars
  - Sleek dark mode visual aesthetics
  - Butter-smooth animations and transitions
- **Local Storage**: Your data stays firmly on your device.

## 💻 Tech Stack

- **React 18**
- **Vite**
- **Custom Modern CSS**

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/selvaganapathycoder/Habit-Tracker.git
   cd Habit-Tracker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in Browser:**
   Visit `http://localhost:5173` to view the app!
